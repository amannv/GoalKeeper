### GoalKeeper

GoalKeeper is a full-stack goal management application built with the MERN stack. It allows users to create, track, and manage their daily goals through a clean and intuitive interface.

This project helped me understand how modern web applications are built from frontend to backend, including API integration, authentication, database management, and deployment.

### Live Demo

[https://goal-keeper-eight.vercel.app](https://goal-keeper-eight.vercel.app)

### About the Project

GoalKeeper helps users stay organized by providing a simple platform to manage daily goals and tasks.

The main focus of this project was to gain hands-on experience with full-stack development and understand how the frontend and backend communicate in a real application.

### Features

1. User authentication

2. Create and manage personal goals

3. View all saved goals

4. Responsive design for mobile and desktop devices

5. Clean and user-friendly interface

6. Full-stack architecture with separate frontend and backend

### Tech Stack

| Layer      | Technologies           |
| ---------- | ---------------------- |
| Frontend   | React.js, Tailwind CSS |
| Backend    | Node.js, Express.js    |
| Database   | MongoDB                |
| Deployment | Vercel                 |
